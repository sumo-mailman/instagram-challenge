# User Stories

## MVP 
```
As a social media user
So that I can share my life updates
I would like to be able to post an image to my feed 
```
```
As a social media user
So that I can tell others what I think about their photos 
I would like to be able to comment on someones image 
```
```
As a social media user
So that I can tell others I love their photos 
I would like to be able to like a picture 
```
```
As a social media user
So that I can post my own photo 
I would like to be able to sign in
```
```
As a social media user
So that others people can't post as me
I would like to be able to sign out 
```
```
As a user who bitterly regrets a post
So that I can change my mind at a later date
I would like to remove a post I made from the feed

```
